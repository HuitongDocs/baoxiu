## 创建我的单位

可选择小程序创建或者web端（https://baoxiu.larkea.com/）创建，两者数据互通；

两者都需填写单位名称，单位简介和单位类型；

不同的是，小程序可选择单位的地区，web端可上传公司的Logo以及扫码报修和加入单位的权限

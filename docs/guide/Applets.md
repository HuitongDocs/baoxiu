## 任务功能

::: warning 注意
任务功能只有工程师以上的级别能查看和操作
:::


| 功能点       | 详细说明                                                     |
| ------------ | ------------------------------------------------------------ |
| 任务查询     | 首页任务栏页面下的任务都是已经分配给当前工程师的任务         |
| 维修任务认领 | 在我的页面的报修池下，点击待受理的任务再点击我来维修来认领任务 |
| 任务协同     | 需要多个工程师共同维修的情况，领取任务后，点击更多操作，再点击协同，然后选择想要报送的团队或个人 |
| 任务转派     | 转派成功后，任务与当前工程师无关，领取任务后，点击更多操作，再点击转派，然后选择想要报送的团队或个人 |
